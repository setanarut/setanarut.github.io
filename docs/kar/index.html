Controls

  • Movement/Look/Target: [WASD]
  • Jump: Space
  • Run: RightShift
  • Place Block: Left Arrow

    • Note: If there is a player or item at the block location to be placed, the block will not be placed.

  • Break Block: Right Arrow
  • Drop selected slot item: Up Arrow
  • Select next slot : E
  • Select previous slot : Q
  • Delete selected slot : Backspace
